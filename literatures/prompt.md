/idea-discovery "Build a digital on-chip power model that can be implemented in the processor core (with RTL), and inference the power consumption of the processor at the runtime when processor runs different benchmarks. The on-chip power model should be built with linear model or tree-based model to minimize the area and power overhead. However, you should propose an automatic feature engineering procedure to select the most important RTL signals as the input features of the power model. You may consider using to embed the infomation of the raw RTL signals before performing the model fitting. You should carefully consider the correlation between different RTL signals since mutiple signals may need to work together to affect the power. Some possible approaches including using tree-based feature selection techniques, use ViF or Pearson coefficient to indicate eliminate the highly correlated signals, use attention layer to automatically identify the cross-correlation between signals, use dimensionality reduction methods, or building a hierarchical model that captures the interactions between different RTL signals. Note that if the feature engineering cannot be purely offchip (like you need to implement the embedding layer on-chip), you should consider the area and power overhead of implementing this feature engineering procedure on-chip as well."

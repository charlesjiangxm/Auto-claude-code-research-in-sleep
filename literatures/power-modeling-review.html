On-Chip Power Proxy & Estimation — Related Works

31 unique papers sorted by publication year (oldest → newest), filtered from local PDFs plus the original review list. Internet links are provided for every entry, and local-copy links are shown when a local PDF was found.

Paper Name Year Link Method Main Contribution
1Register-Transfer Level Estimation Techniques for Switching Activity and Power Consumption
 1996 IEEE / Local Copy Piecewise-Linear + Bit-Level Glitch Modeling
Builds RTL block power models and estimates both word-level and bit-level glitching activity, then feeds those signal statistics into block-level power models.		 Showed that RTL power estimation can explicitly model glitching in datapath and control logic, producing estimates within 7% of final gate-level analysis.
2Power Modeling for High-Level Power Estimation
 2000 DOI / Local Copy 4D Table-Based Macromodel
Automatically characterizes a combinational block with a four-dimensional table indexed by input probability, transition density, spatial correlation, and output zero-delay transition density.		 Established a practical high-level power macromodel with good accuracy across wide input statistics, making early design-space exploration far cheaper than repeated low-level simulation.
3Runtime Power Monitoring in High-End Processors: Methodology and Empirical Data
 2003 IEEE / PDF / Local Copy Piecewise / Per-Unit Linear Regression
Divides the processor into functional units and correlates per-unit PMC activity counts with measured power using linear weights.		 Foundational work showing real-time power can be accurately estimated from existing hardware counters. Introduced "power phases" and per-unit power decomposition, enabling fine-grained thermal and power management.
4Power Prediction for Intel XScale Processors using Performance Monitoring Unit Events
 2005 IEEE / ACM DL / Local Copy Linear Regression across DVFS States
Identifies PMCs as proxies for activity factors and predicts power across different voltage/frequency operating points.		 Validated the PMC-based power proxy approach for low-power/embedded processors and demonstrated modeling across Dynamic Voltage and Frequency Scaling states.
5Parameterized Logic Power Consumption Models for FPGA-Based Arithmetic
 2005 IEEE / Local Copy Parameterized Word-Level Models
Uses word-level signal statistics to estimate the logic power of FPGA arithmetic blocks such as adders and multipliers without low-level simulation.		 Delivered fast high-level FPGA arithmetic power estimates within about 10% of XPower, making arithmetic design-space exploration much more practical.
6Complete System Power Estimation Using Processor Performance Events
 2007 IEEE (conf.) / IEEE (journal) / Local Copy Linear Regression on PMCs
Selects a small set of highly correlated Processor Performance Events and models power as a linear combination of their counts.		 Demonstrated that a small, generic set of software-visible performance counters can accurately estimate total system power (CPU + DRAM) for commodity processors without custom hardware.
7McPAT: An Integrated Power, Area, and Timing Modeling Framework for Multicore and Manycore Architectures
 2009 ACM DL / Local Copy Analytical Architecture Model
Combines component-level analytical models for cores, caches, NoCs, memory controllers, area, and timing behind an XML interface for architectural simulators.		 Became a standard architecture-level framework for jointly evaluating power, area, and timing trade-offs in multicore and manycore design exploration.
8RAPL: Memory Power Estimation and Capping
 2010 IEEE / ACM DL / Local Copy Analytical Event-Based Modeling
Assigns energy weights to specific transaction types (e.g., DRAM reads/writes) and accumulates them in real-time within hardware/microcode.		 Describes the industry-standard Intel RAPL interface. Demonstrated how to implement a power proxy with negligible software overhead by embedding accumulator logic directly into hardware/firmware for closed-loop power management.
9An Integrated GPU Power and Performance Model
 2010 ACM DL / Local Copy Statistical Instruction Analysis
Estimates GPU power by analyzing PTX instruction rates and global memory accesses, applying empirical weights from micro-benchmarks.		 Extended PMC-based power proxies to massively parallel GPU architectures, highlighting the need to account for memory coalescence and warp divergence in non-CPU IP blocks.
10Decomposable and Responsive Power Models for Multicore Processors using Performance Counters
 2010 ACM DL / Local Copy PCA + Statistical Clustering
Analyzes all counter behaviors to automatically group and select those capturing power-consumption variance with minimal redundancy.		 Proposed an automated methodology for optimal proxy selection, removing reliance on expert intuition and maximizing accuracy while minimizing model complexity.
11PrEsto: An FPGA-Accelerated Power Estimation Methodology for Complex Systems
 2010 IEEE / Local Copy Linear Regression on I/O Boundary Signals
Selects input/output signals at module boundaries as proxies and uses linear regression, integrated into FPGA-based performance simulators.		 Predicts cycle-by-cycle power of LEON3 and ARM Cortex-A8 cores within 6% of commercial gate-level tools while running orders of magnitude faster via FPGA acceleration.
12Accurate Fine-Grained Processor Power Proxies
 2012 IEEE / ACM DL / Local Copy Weighted Counter-Based Architecture
Monitors specific on-chip activities, assigns power-use weight factors, and aggregates weighted counts into proxy values.		 Presented an accurate runtime per-core power proxy enabling fine-grained microprocessor power management and what-if analysis by the energy-efficiency controller. Validated at both chip and core levels.
13An Accurate Dynamic Power Model on FPGA Routing Resources
 2012 IEEE / Local Copy RC-Based Routing Power Model
Models pass-transistor chains and multiplexers with a more accurate RC formulation that tracks varying input and output voltages in FPGA routing resources.		 Improved dynamic FPGA routing-power accuracy to below 3% for pass-transistor chains and below 5% for muxes versus HSPICE simulation.
14Dynamic and Leakage Power Estimation in Register Files Using Neural Networks
 2012 DOI / Local Copy Neural Network Register-File Model
Trains neural networks on register-file structural and usage parameters to estimate both dynamic and leakage power without transistor-level analysis.		 Extended ML-based estimation beyond total dynamic power by modeling register-file leakage and dynamic components together for early embedded-system design.
15Method for Dynamic Power Monitoring on FPGAs
 2014 HAL / Local Copy Automatic Signal Selection + Event Counters
Selects relevant RTL signals with a Greedy Stepwise filter, inserts event counters, and automatically derives a runtime FPGA power model.		 Presented a generic CAD flow that auto-generates runtime power monitors for FPGA SoCs while balancing estimation accuracy against monitoring overhead.
16Early Stage Real-Time SoC Power Estimation Using RTL Instrumentation
 2015 IEEE / Local Copy SVD-Based Model Abstraction
Uses singular value decomposition to abstract principal components of the register-toggling-to-power relationship, then instruments the model into RTL for FPGA-based real-time estimation.		 Cycle-by-cycle power prediction within 5% accuracy on three IP cores. Addresses the challenge of real-time, long-interval accurate estimation for system-level software/hardware tuning.
17Power Prediction of Embedded Scalar and Vector Processor: Challenges and Solutions
 2017 IEEE / Local Copy Selective Event Regression
Builds a canonical dynamic-power equation from carefully chosen PMCs, stall probe points, and instruction groups for a commercial embedded processor.		 Documented a production-ready methodology for PMC-based power prediction, reaching about 8% average error versus commercial EDA power while greatly reducing compute cost.
18PowerProbe: Run-time Power Modeling Through Automatic RTL Instrumentation
 2018 IEEE / Local Copy Automatic RTL Instrumentation
Extracts a power model from the RTL description without leveraging any performance counter. Post-map logic simulation provides power traces used to derive and instrument the model.		 Trades power-model accuracy against monitoring resource cost. The automatic instrumentation strategy ensures timely implementation regardless of target platform complexity.
19KAPow: High-Accuracy, Low-Overhead Online Per-Module Power Estimation for FPGA Designs
 2018 ACM DL / Local Copy Adaptive Per-Module Online Modeling
Combines register-level switching activity with system-level power measurements, then refines a live per-module model online with automated signal selection and pruning.		 Enabled module-level online FPGA power breakdowns with low overhead, supporting power-aware task mapping and adaptive optimization instead of whole-chip estimates only.
20An Ensemble Learning Approach for In-Situ Monitoring of FPGA Dynamic Power
 2018 IEEE / Local Copy Decision-Tree Ensemble Monitor
Uses a CAD flow to select features, tune hyperparameters, train multiple customized tree learners, and synthesize the estimator into FPGA hardware.		 Improved runtime FPGA dynamic-power prediction to within about 1.9% maximum error while keeping LUT overhead near 1.22% of the target FPGA.
21Simmani: Runtime Power Modeling for Arbitrary RTL with Automatic Signal Selection
 2019 ACM DL / Local Copy K-Means Clustering + Regression
Clusters RTL signals by toggle-pattern similarity, selects representative signals, and constructs a workload-independent activity-based power model.		 Automatically instruments FPGA-accelerated RTL simulations with runtime activity counters. Demonstrated on a heterogeneous processor running real-world ML workloads, not just microbenchmarks.
22PRIMAL: Power Inference using Machine Learning
 2019 ACM DL / Local Copy ML Models (Ridge, GBT, MLP, CNN)
Trains ML models on design verification testbenches using transitions of all registers as features, characterizing power of reusable circuit building blocks.		 Average power error <1% across diverse benchmarks (15× faster than commercial RTL tools). Cycle-by-cycle estimation 50× faster than gate-level analysis with <5% error.
23GRANNITE: Graph Neural Network Inference for Transferable Power Estimation
 2020 IEEE / ACM DL / Local Copy Graph Neural Network (GNN)
Represents netlists as graphs; learns to propagate average toggle rates through combinational logic. Infers toggle rates on new workloads/netlists from RTL simulation in seconds.		 >18.7× speedup over gate-level simulation with <5.5% error. Key novelty is transferability — a trained model applies to previously unseen netlists with 25.9% lower error than probabilistic approaches.
24NeuPow: A CAD Methodology for High-Level Power Estimation Based on Machine Learning
 2020 ACM DL / Local Copy ANN Propagation Across Component Models
Represents each digital component with an ANN power model and propagates signals through connected ANN blocks, including a frequency-aware variant.		 Showed that ML component models can scale to composite systems with less than 9% relative error and roughly 84x speedup over conventional power-estimation flows.
25McPAT-Calib: A Microarchitecture Power Modeling Framework for Modern CPUs
 2021 IEEE / Local Copy Black-Box Calibration (XGBoost / Lasso)
Takes the McPAT architectural power simulator and calibrates it against real hardware PMCs using machine learning.		 Bridges pre-silicon simulation and post-silicon runtime proxies by correcting theoretical models to match runtime reality, yielding physically meaningful proxies rather than purely statistical correlations.
26APOLLO: An Automated Power Modeling Framework for Runtime Power Introspection in High-Volume Commercial Microprocessors
 2021 ACM DL / Local Copy MCP-Based Proxy Selection
Automatically selects a tiny set of RTL power proxies with minimax concave penalty (MCP) feature selection, then builds a per-cycle estimator and synthesizable on-chip power meter.		 Demonstrated per-cycle power modeling and runtime OPM integration for industrial Arm CPUs with strong accuracy and under 1% area/power overhead.
27DEEP: Developing Extremely Efficient Runtime On-Chip Power Meters
 2022 ACM DL / Local Copy MCP Coarse Selection + Bit-Level Refinement
Selects OPM inputs from individual RTL signal bits. A two-step proxy selection (coarse → fine) feeds a linear model that sums weights of toggling signals.		 Reduced OPM hardware cost by 4–6× over prior art while achieving R² = 0.97 in per-cycle power prediction with unprecedented low area overhead.
28PANDA: Architecture-Level Power Evaluation by Unifying Analytical and Machine Learning Solutions
 2023 arXiv / Local Copy Hybrid Analytical + ML Model
Unifies classical analytical architecture models and ML calibration so the estimator can generalize to unseen designs and even new process nodes with limited training data.		 Bridged the gap between interpretable analytical models and data-driven accuracy, improving architecture-level power prediction for unknown designs under realistic data scarcity.
29PROPHET: Predictive On-Chip Power Meter in Hardware Accelerator for DNN
 2023 IEEE / Local Copy Data-Pattern Predictive OPM
Samples predefined data patterns during memory access to predict accelerator power before the corresponding MAC activity actually occurs.		 Introduced a predictive DNN-accelerator power meter that reports power dozens of cycles in advance with 4-cycle resolution and under 2% area overhead.
30Data-Pattern-Based Predictive On-Chip Power Meter in DNN Accelerator
 2024 IEEE / Local Copy Data-Pattern Predictive Power Model
Extends PROPHET into a data-pattern-based power-modeling flow for MAC-based DNN accelerators across multiple dataflows and data types.		 Validated predictive DNN-accelerator power metering at 2-cycle resolution with R² > 0.9, normalized MAE below 7%, and under 4.5% area/power overhead.
31Run-Time Power Management System by On-Die Power Sensor with Silicon Machine Learning-Based Calibration in a 3nm Octa-Core CPU
 2025 IEEE / Local Copy On-Die Sensor + Silicon ML Calibration
Computes runtime power from synchronized on-die current and voltage sensors, then calibrates sensing accuracy with a silicon-trained ML model.		 Showed a production 3nm CPU power-sensing system with 94.83% power accuracy, 10µs monitoring windows, and only 0.0313% CPU-area overhead.
